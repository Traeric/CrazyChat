# CrazyChat
这是一个小型的聊天跟社交系统

# 使用技术
## 前端
* vue
* nuxt
* element ui
* axios
* kindeditor
* stylus
## 后端
* springboot
* springmvc
* spring data jpa
* spring data redis
* spring data mongo
* spring security
* spring cloud
* twitter的雪花算法(计算分布式id)
* jwt
# 数据库
* mariadb   
* redis    
* mongodb
